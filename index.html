Privacy Policy

Effective Date: Jan 28, 2026

SlyOS is designed with a "Zero-Knowledge" architecture. We provide the infrastructure (SDKs, Model Weights, Telemetry APIs) but we do not process the actual inputs (prompts) or outputs (completions) of your users.

Data Collection

  • Device Specs: RAM, GPU type, and OS version to optimize model selection.
  • Performance Metrics: Latency (ms), Tokens per second (t/s), and Error rates.

No Content Access

Inference runs locally on the end-user's hardware. SlyOS servers never receive the text data generated by the models.

Installation

Install the SlyOS SDK to begin deploying models to the edge.

$ npm install @slyos/sdk

Quick Start

Initialize the client with your organization key.

import { Sly } from '@slyos/sdk';

const sly = new Sly({
  apiKey: 'sk_live_...'
});

// Downloads and caches the model automatically
await sly.load('quantum-360m');
